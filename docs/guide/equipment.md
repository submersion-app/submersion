# Equipment Management

Track all your diving gear, monitor service schedules, and never miss a maintenance deadline.

## Equipment Catalog

The **Gear** tab shows all your equipment with status indicators.

### Status Indicators

| Status | Color | Meaning |
|--------|-------|---------|
| **Active** | Green | Good to use |
| **Needs Service** | Yellow | Service due soon |
| **Overdue** | Red | Service overdue |
| **In Service** | Blue | Currently being serviced |
| **Retired** | Gray | No longer in use |

## Adding Equipment

1. Tap **+ Add Gear** from the Gear tab
2. Fill in the equipment details
3. Set service interval if applicable
4. Save

### Equipment Fields

| Field | Description |
|-------|-------------|
| **Name** | Your name for this item |
| **Type** | Category (see types below) |
| **Brand** | Manufacturer |
| **Model** | Product model |
| **Serial Number** | For warranty/service |
| **Size** | S, M, L, XL, or specific |
| **Status** | Current status |
| **Purchase Date** | When you bought it |
| **Purchase Price** | Cost and currency |
| **Last Service Date** | Most recent service |
| **Service Interval** | Days between services |
| **Notes** | Additional info |

## Service Tracking

### Setting Service Intervals

Common service intervals:

| Equipment | Typical Interval |
|-----------|------------------|
| Regulator | 1 year / 100 dives |
| BCD | 1 year |
| Computer | Battery dependent |
| Cylinder | Visual: 1 year, Hydro: 5 years |

### Service Reminders

When service is due:
- Yellow badge appears 30 days before
- Red badge when overdue
- Notification in equipment list

### Recording Service

1. Open the equipment detail page
2. Tap **Service History**
3. Tap **+ Add Service**
4. Enter service details

Service record fields:
- **Service Type** - Annual, Repair, Inspection, etc.
- **Date** - When serviced
- **Provider** - Shop or technician
- **Cost** - Service cost
- **Next Due** - When next service needed
- **Notes** - What was done

## Equipment Sets

Group equipment into sets for quick selection:

### Creating a Set

1. Go to **Gear** > **Sets**
2. Tap **+ Create Set**
3. Name the set (e.g., "Tropical", "Cold Water")
4. Select equipment items
5. Save

### Using Sets

When logging a dive:
1. Go to the Equipment section
2. Tap **Apply Set**
3. Select a set
4. All items are linked to the dive

## Per-Dive Tracking

### Linking Equipment

For each dive, you can link the equipment you used:

1. In the dive entry form, find **Equipment**
2. Tap **Add Equipment**
3. Select items from your catalog
4. Repeat for all gear used

### Benefits

- Track equipment usage over time
- Know which gear was used when issues occurred
- See total dives per equipment item

## Weight Calculator

The built-in weight calculator helps determine proper weighting:

1. Go to **Gear** > **Weight Calculator** (or Tools)
2. Select your exposure suit type and thickness
3. Select tank type
4. Select water type (salt/fresh)
5. View recommended weight

## Equipment Tips

### Organization

- Name items clearly: "Primary Perdix" vs just "Computer"
- Add serial numbers for warranty claims
- Take photos of equipment (feature planned)

### Service Records

- Keep receipts attached to service records (notes)
- Track cumulative service costs
- Note any issues found during service

### Retirement

When equipment is no longer usable:
1. Set status to **Retired**
2. Add notes explaining why
3. Item remains in history but not active

### Loaned Equipment

Track borrowed/loaned gear:
1. Set status to **Loaned**
2. Add note with who has it
3. Update when returned
