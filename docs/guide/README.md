## Quick Overview

### The Main Tabs

Submersion has five main sections accessible from the bottom navigation bar (mobile) or side rail (desktop):

| Tab | Purpose |
|-----|---------|
| **Dives** | Your dive log - view, add, and edit dives |
| **Sites** | Dive site database with maps |
| **Gear** | Equipment catalog and service tracking |
| **Stats** | Analytics and personal records |
| **Settings** | App configuration and preferences |

### Core Workflow

1. **Create a diver profile** - On first launch, you'll set up your diver profile
2. **Add dive sites** - Build your site database (or import existing sites)
3. **Log dives** - Manually or download from a dive computer
4. **Track gear** - Add your equipment with service reminders
5. **Analyze stats** - Review your diving patterns and records
