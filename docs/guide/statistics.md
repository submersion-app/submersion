# Statistics & Analytics

Submersion provides eleven specialized dashboards for deep insights into your diving history.

## Statistics Dashboard

The **Stats** tab is your analytics hub with multiple specialized pages.

## Overview Stats

The main dashboard shows at-a-glance metrics:

| Metric | Description |
|--------|-------------|
| **Total Dives** | Lifetime dive count |
| **Total Bottom Time** | Cumulative dive time |
| **Max Depth** | Deepest dive ever |
| **Sites Visited** | Unique dive sites |
| **Countries** | Countries dived in |

## Charts & Visualization

### Interactive Charts

All charts support:
- **Tap** - View data point details
- **Pinch** - Zoom in/out
- **Drag** - Pan across data
- **Legend** - Toggle data series

### Chart Types

| Type | Used For |
|------|----------|
| **Line** | Trends over time |
| **Bar** | Comparisons |
| **Pie** | Proportions |
| **Histogram** | Distributions |
| **Heat Map** | Time patterns |

## Filtering Stats

### Date Range

Filter statistics by:
- **All Time** - Complete history
- **This Year** - Current year
- **Last 12 Months** - Rolling year
- **Custom Range** - Any date range

### Other Filters

Combine filters for specific analysis:
- By dive site
- By dive type
- By buddy
- By equipment

## Exporting Stats

### PDF Report

Generate a printable statistics report:

1. Go to Stats > **Export**
2. Select **PDF Report**
3. Choose date range
4. Download PDF

### Data Export

Export raw data for external analysis:

1. Go to Stats > **Export**
2. Select **CSV Data**
3. Choose what to include
4. Download CSV

## Tips for Using Statistics

### Regular Review

- Check stats monthly to track progress
- Compare year-over-year trends
- Set goals based on historical data

### Data Quality

Good statistics require good data:
- Log dives promptly and accurately
- Include all relevant fields
- Use consistent terminology

### Insights

Use stats to:
- Identify favorite conditions
- Plan future trips
- Track safety metrics (ascent rates)
- Monitor equipment usage
