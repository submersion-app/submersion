# Tags & Dive Types

Organize your dives with custom colored tags and dive type classifications.

## Tags

### What Are Tags?

Tags are custom labels you create to categorize dives:
- Flexible organization
- Color-coded for visibility
- Multiple tags per dive
- Personal to each diver

### Creating Tags

1. Go to **Settings** > **Tags**
2. Tap **+ Create Tag**
3. Enter tag name
4. Choose color
5. Save

### Tag Colors

20 preset colors available:
- Red, Orange, Yellow, Green, Blue
- Purple, Pink, Teal, Cyan, Lime
- Amber, Deep Orange, Indigo, Brown, Gray
- And more

### Applying Tags

1. In dive entry, find **Tags** section
2. Tap to open tag selector
3. Select one or more tags
4. Tags appear on dive

### Multiple Tags

Dives can have any number of tags:
- "Night Dive" + "Photography"
- "Training" + "Navigation"
- "Wreck" + "Deep" + "Sidemount"

## Filtering by Tags

### In Dive List

1. Tap filter icon
2. Select **Tags**
3. Choose tag(s)
4. View matching dives

### Combining Filters

Combine with other filters:
- Tags + Date range
- Tags + Dive type
- Tags + Location

### Selecting Dive Type

1. In dive entry, find **Dive Type**
2. Tap to open selector
3. Choose type
4. Saved with dive

### Custom Dive Types

Create your own types:

1. Go to **Settings** > **Dive Types**
2. Tap **+ Add Type**
3. Enter name
4. Save

Custom types appear alongside built-in types.

## Tags vs. Dive Types

### Key Differences

| Aspect | Tags | Dive Types |
|--------|------|------------|
| **Per Dive** | Multiple | One |
| **Purpose** | Flexible categorization | Primary classification |
| **Stats** | Tag-based filtering | Type-based statistics |
| **System** | User-created | Built-in + custom |

### Using Together

Best practice:
- **Dive Type** = Primary category (Wreck)
- **Tags** = Additional aspects (Night, Photography, Deep)

## Organization Strategy

### Keep It Simple

Start with few tags:
- Add as needed
- Don't over-categorize
- Review periodically

### Consistent Use

Be consistent:
- Same tags for same purposes
- Update old dives if needed
- Train yourself to tag

### Periodic Review

Maintain your tags:
- Delete unused tags
- Merge similar tags
- Update colors for visibility

## Editing Tags

### Rename Tag

1. Go to **Settings** > **Tags**
2. Tap on tag
3. Edit name
4. Save

### Change Color

1. Open tag settings
2. Tap color
3. Select new color
4. Save

### Delete Tag

1. Go to **Settings** > **Tags**
2. Swipe left on tag
3. Tap **Delete**
4. Tag removed from all dives

## Searching by Tags

Use search to find tagged dives:
- Type tag name in search
- Results include matching dives
- Combine with other search terms

## Import/Export

### Tag Data

Tags export with dives:
- UDDF includes tag info
- CSV can include tags
- Imports create tags as needed

### Cross-Device

If using cloud sync:
- Tags sync across devices
- Color settings preserved
- Per-diver tag sets
