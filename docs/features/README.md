# Features Overview

Submersion includes professional-grade features for all levels of divers.

## Feature Highlights

### For Recreational Divers

| Feature | Description |
|---------|-------------|
| [**Dive Logging**](guide/dive-logging.md) | Comprehensive dive entries with 40+ fields |
| [**Dive Sites**](guide/dive-sites.md) | GPS-enabled site database with maps |
| [**Equipment**](guide/equipment.md) | Gear tracking with service reminders |
| [**Statistics**](guide/statistics.md) | Personal records and analytics |
| [**Buddies**](features/buddies.md) | Track dive partners and certifications |
| [**Marine Life**](features/marine-life.md) | Species sighting database |

### For Technical Divers

| Feature | Description |
|---------|-------------|
| [**Profile Analysis**](features/profile-analysis.md) | Interactive depth/pressure/temp charts |
| [**Decompression**](features/decompression.md) | Buhlmann ZH-L16C algorithm |
| [**O2 Tracking**](features/oxygen-tracking.md) | CNS%, OTU, ppO2 monitoring |
| [**Multi-Gas**](features/multi-gas.md) | Air, nitrox, trimix support |
| [**Multi-Computer**](guide/dive-computer.md) | Multiple profiles per dive |

### Organization Tools

| Feature | Description |
|---------|-------------|
| [**Trips**](features/trips.md) | Group dives into trips |
| [**Tags**](features/tags.md) | Custom colored tags |
| [**Dive Types**](features/tags.md#custom-dive-types) | Built-in and custom categories |
| [**Search**](guide/dive-logging.md#search) | Full-text search across all data |

## Version History

### v1.5 (Current)

Technical diving features:
- Dive computer integration (300+ models)
- Buhlmann ZH-L16C decompression algorithm
- CNS% and OTU oxygen toxicity tracking
- Multi-profile support
- Weather and tide integration

### v1.1

Enhanced mapping and organization:
- GPS capture and reverse geocoding
- Interactive maps with clustering
- Tag system with colors
- Profile zoom and pan
- Favorites

### v1.0

Core functionality:
- Dive logging with all fields
- Dive site database
- Equipment management
- Statistics dashboards
- UDDF/CSV import/export
- Multi-diver support
