# Trips

Organize dives into trips for better organization and trip-level statistics.

## What Is a Trip?

A trip groups related dives together:
- Multi-day dive vacation
- Liveaboard expedition
- Weekend dive trip
- Training course

## Creating Trips

### From Settings

1. Go to **Settings** > **Trips**
2. Tap **+ Add Trip**
3. Fill in details
4. Save

### From Dive Entry

When logging a dive:
1. Find **Trip** field
2. Tap to select or create
3. Link dive to trip

## Trip Fields

| Field | Description |
|-------|-------------|
| **Name** | Trip name/title |
| **Start Date** | When trip begins |
| **End Date** | When trip ends |
| **Location** | General area |
| **Resort Name** | If resort-based |
| **Liveaboard Name** | If liveaboard |
| **Notes** | Trip details |

## Trip Types

### Resort Trip

For shore-based diving:
- Set **Resort Name**
- Location = resort location
- Add multiple dive sites

### Liveaboard

For live-aboard trips:
- Set **Liveaboard Name**
- Location = cruising area
- Track multiple sites visited

### Day Trip

For single-day excursions:
- Same start/end date
- Location = departure point
- Quick organization

## Linking Dives

### Manual Linking

1. Open dive entry
2. Find **Trip** dropdown
3. Select the trip
4. Save

### Bulk Linking

For existing dives:
1. Go to dive list
2. Select multiple dives
3. Choose **Add to Trip**
4. Select trip

### Auto-Suggestion

When creating a dive:
- If date falls within a trip
- Trip is suggested automatically
- Tap to confirm or change

## Trip Organization

### Naming Conventions

Clear trip names help:
- "Palau November 2024"
- "Red Sea Liveaboard - MY Blue"
- "Kona Big Island Week"

### Date Accuracy

Set accurate dates:
- Include travel days or not (your choice)
- Helps with auto-suggestions
- Better historical tracking

## Viewing Trips

### Trip List

Access all trips:
1. Go to **Settings** > **Trips**
2. View sorted by date
3. Filter by year

### Trip Filtering

In dive list:
1. Tap filter icon
2. Select **By Trip**
3. Choose a trip
4. View only those dives

## Trip vs. Tags

### When to Use Trips

- Specific time period
- Geographic boundary
- Start/end dates matter
- Want trip statistics

### When to Use Tags

- Ongoing categorization
- Cross-cutting themes
- No specific dates
- Multiple applies

### Using Both

You can combine:
- Trip: "Cozumel March 2024"
- Tags: "Night Dive", "Cenote", "Training"

## Export Options

### Trip Export

Export all trip dives:
1. Open trip detail
2. Tap **Export**
3. Choose format
4. All trip dives included

### Trip Report

Generate trip summary:
- PDF with all dives
- Statistics summary
- Site list
- Species sightings

## Planning Future Trips

### Placeholder Trips

Create trips before traveling:
1. Add trip with future dates
2. Link dives as you log them
3. Easy organization

### Pre-Trip Info

Use notes for:
- Flight information
- Accommodation details
- Emergency contacts
- Rental gear notes
- Visa requirements

## Trip Best Practices

1. **Create early** - Set up trip before you go
2. **Link promptly** - Add dives daily
3. **Complete after** - Add summary notes
4. **Review stats** - See trip performance
5. **Export backup** - Save trip data
